id="ckepop"></div></h6> </div> <div class="clear"></div> <dd class="con"> <p>Admin5访谈后记：A5的豆豆对我进行了访谈，这是A5对我进行的第二次访谈（点此查看第一次访谈：<a target="_blank" href="http://lusongsong.com/reed/219.html">就这样从草根到知名的转变</a>），没想到还顶置在A5首页，这也许才是草根的体现。豆豆，一个很可爱的小姑娘，在A5负责访谈栏目和招聘板块，访谈时间花了近四个小时。我是直接转载豆豆发布访谈，本来我说的很俗，豆豆发布时美化了一下，部分内容我就不转载了，我只转载精华部分，只是想告诉大家：做一个网站很简单，坚持、勤奋的做下去，你也能成功。</p><p><span style="color: rgb(0, 0, 255);">豆豆：</span>首先和新老朋友打个招呼吧，简单的介绍一下自己。</p><p><span style="color: rgb(255, 0, 0);">卢松松：</span>呵呵，大家好，我是卢松松，很荣幸又和A5的朋友见面了，每次来总有不同的收获，同样对运营和营销也有着不同的理解!</p><p>我的<a title="互联网" href="http://lusongsong.com/tags/hulianwang.html" target="_blank" class="keywordlink">互联网</a>之路开始于我的大学时期，一个舍友当时兼职当地一家论坛版主，在他的影响下，我第一次接触到论坛管理，从此我就踏上了这条“不归路”。</p><p>在后来又了解到这个论坛发展成公司的<a title="创业" href="http://lusongsong.com/tags/chuangye.html" target="_blank" class="keywordlink">创业</a>经历，从那时起，我对互联网的喜爱从游戏转到创业。随后我参加了电子科技大学的大学生科技节活动，做了个静态的HTML网页，竟然获得了二等奖，从此走向了网站建设道路，建了“校园闹翻天论坛”和一个源码交易站，当然还有一个技术笔记博客，也就是博客前身。</p><p><span style="color: rgb(0, 0, 255);">豆豆：</span>据豆豆了解，目前Alexa最新排名万名内的，除了月光就是松松，豆豆在这恭喜松松也祝博客越来越火，近期有什么新的动向或是打算?</p><p><span style="color: rgb(255, 0, 0);">卢松松：</span>谢谢哈，借您吉言，希望以后越来越像个网站。对于博客而言，我的想法就是维持现状，坚持和博友之间的互动。</p><p>关于我自己，近期的主要工作重心就是和我烧网联合搞一个“大博客”计划，目的很简单，就是增加部分博主收益，提升大部分独立博客流量，实现共赢。</p><p><span style="color: rgb(0, 0, 255);">豆豆：</span>做过论坛、建过网站、创过业，现在打工，业余管理博客，我相信好多站长都会问这样一个问题，你的技术和网络运营都这么娴熟，为什么不自己做个站好好运营，也许比现在更火?</p><p><span style="color: rgb(255, 0, 0);">卢松松：</span>这个问题很多人问过我，我自己也想过。可能还是和我经历有关，开公司那一年中，接触了不少人，可以说是五花八门，见过最多的就是做了几十页项目书外加一个刚做好的网站就要拉投资人，我觉得不靠谱，我喜欢脚踏实地的做事。为了做站而做站，不是我目的，通过网站这个平台最终你想获得什么，是名还是利?</p><p>听过不少做网站年入几十万上百万的人，大家蜂拥而至。最后这些人慢慢堆积成山，站在山顶上风光的也就那么几个，多了就站不住。那时做网站业务时就想明白了，同样为了挣钱，怎么挣都是挣，没必要拐个弯挣钱。我做博客初衷也不是为了出名，当时把网站卖了后，心想一个搞网站的人，连个自己的站都没有很不合适，就把这博客独立出来，给客户看看案例，写写心得什么的。</p><p><span style="color: rgb(0, 0, 255);">豆豆：</span>就你自己对博客的了解，目前的博客主要分为哪几类?个人比较看好哪种类型?</p><p><span style="color: rgb(255, 0, 0);">卢松松：</span>从大的分类来说就2类，BSP博客和独立博客，先说说独立博客，这类博客能搭建起来，大多都是有一定网站技术的人，所以他们的博客主要是讨论SEO、IT评论、营销一类的，但因政策因素和技术水平限制，国内很难有Engadget、ReadWriteWeb这样的博客。</p><p>至于BSP博客也就是注册博客，如新浪、网易的博客。这种非技术类博客更容易收到媒体、门户、朋友的关注，通常电视台关注的都是这类博客，现在还是全民娱乐时代。</p><p><span style="color: rgb(255, 0, 0);">豆豆：</span>如何留住访客，提高回访率，让他下次还想再来逛逛?</p><p><span style="color: rgb(255, 0, 0);">卢松松：</span>有问必答，热情相待，你对他好，他自然会常来，同样，他再来的时候，又能学习到新的东西。</p><p><span style="color: rgb(0, 0, 255);">豆豆：</span>对于未来个人博客发展的方向，你有什么看法?豆豆在想，它会不会也电子商务化呢，每人在自己的博客上出售自己的闲置物品或是物物交换，或是网店和博客相结合的模式。</p><p><span style="color: rgb(255, 0, 0);">卢松松</span>：独立博客这个行业其实很窄，未来发展我也看不清，走一步算一步吧，不过我认为未来博客可能会更多参与到互联网的各个角落中，比如或评论或发出自己的声音，我们知道，大网站上很难鉴别真假，只有到个人博客中才能听到真实的声音，像豆豆说的这种情况也有可能，互联网还很年轻，对于它的未来还是未知，只有与时俱进才能跟上他的脚步。</p><p><span style="color: rgb(0, 0, 255);">豆豆：</span>关于行业论坛的推广，不知道松松有什么好的意见。比如豆豆现在负责的a5上线不久的招聘版块，如何去吸引有需求的人来这分享信息，打造成站长行业的人才信息基地?</p><p><span style="color: rgb(255, 0, 0);">卢松松：</span>豆豆的信息，平时就有关注，也加入你的访谈群，看见你经常都会有招聘信息的推广。</p><p>A5是行业站，它和地方站点还不同，一般的地方站都会有信息的收集和分类，招聘也是他们的必配版块，一般都是从本地做起，重点发布当地招聘信息，线下活动也好操作，等有一定经验，在招几个分版版主，在当地找一些招聘单位信息发布，全力为有意向的站长朋友引荐去工作。</p><p>像你们这样的专业的行业站点，定位就是站长，在论坛发布招聘信息的目的不是为了填充论坛内容，也不是为了活跃论坛，因为a5不缺人气，前期做的就是让自己的用户都知道这个版块，收集站长们的求职和招聘信息，只有真正为几十个、几百个甚至上千位站长找到工作时，我觉得你们的人才信息基地也就建成了。</p><p><span style="color: rgb(0, 0, 255);">豆豆：</span>博客的盈利模式单一，挂广告，对于博客盈利的开发有什么好的想法?</p><p><span style="color: rgb(255, 0, 0);">卢松松：</span>这个问题我思考过很久，可想来想去，觉得还是投广告最容易操作，与其一开就做不擅长的其他盈利模式，到不如直接把直投广告做精做细，等哪一天真缺钱时，再考虑其他盈利模式。现在做的大博客计划也是种探索，从个人走向商业的探索，大家不都这么过来的吗?呵呵。</p><p><span style="color: rgb(0, 0, 255);">豆豆：</span>在互联网泡了这么些年，有过失败，有过迷惘，也收获到掌声和鲜花，最后和我们谈谈自己的这么多年的感受和期望吧。</p><p><span style="color: rgb(255, 0, 0);">卢松松：</span>现在的感受就是，做什么都应该淡然去面对，无论是鲜花、掌声、还是骂你的。只要路子没错，该怎么做就闷头做下去，就像我博客评论推广方法一样，无论什么事，只要把一件事重复的、认真的做下去就一定能成功，我很少想未来，我只要过好每个现在就行。</p><p>整理编辑：豆豆；文章删减：卢松松，原文地址：<a target="_blank" rel="external

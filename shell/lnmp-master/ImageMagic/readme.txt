
ImageMagic V6.8.9-10 安装说明

软件用途：图形处理

一、软件安装

1、安装方式
编译安装

2、安装文件
imagickInstall.sh	ImageMagic源码编译安装

3、安装目录
/usr/local

执行文件: /usr/local/bin/convert

MagickWand-config Wand-config 等 执行文件在 /usr/local/bin 目录内

配置文件目录：/usr/local/etc/ImageMagick-6


注：无须设置配置文件

如没有特殊要求，可以使用Yum进行安装，无须进行编译安装。

yum 安装命令：

yum -y install ImageMagick ImageMagick-devel
